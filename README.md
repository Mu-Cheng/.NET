然后就是实现

在C#里，命令空间里已经包含了许多库，我们可以直接调用库模板的数据结构

例如
```
Stack st = new Stack();
Queue qu = new Queue();
```

就能得到一个栈和队列
基本操作有

```
Push();//向栈st压栈
Pop();//弹出栈里第一个元素同时返回出来
Peek();访问第一个元素
```
队列

```
Enqueue();//入队
Dequeue();//出队
```


**1.**能实现标准计算器的功能，如计算1+2*90+(23*2/3)


**2.**做了一些输入检验，自动禁止不合法输入
